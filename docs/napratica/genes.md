# Editando GENES com Python 
### Python Aplicado à Bioinformática

Neste vídeo, resolvemos um problema de Bioinformática que nos foi trazido por duas alunas do curso Python do Jeito Certo. O objetivo do programa é abrir arquivos FASTA, que contém descrições de sequências genéticas e fazer a edição dessas sequências com base na quantidade de caracteres '-' (traço) encontrados. A sequência genética editada deve ser escrita em um novo arquivo FASTA.
